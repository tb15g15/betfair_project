# betfair_project

This project is focussed on constructing an agent that acts on the Betfair Exchange to perform trades given some specified criteria derived from previous market data.

<u> __Data__ </u>
The previous market data is taken from the Betwise Smartform Racing Database : https://www.betwise.co.uk/smartform.

<u> __Packages__ </u> 
- ```betfairlightweight```
- ```sqlalchemy```
- ```pymysql```
- ```pandas```
- ```numpy```
- ```scikit```
